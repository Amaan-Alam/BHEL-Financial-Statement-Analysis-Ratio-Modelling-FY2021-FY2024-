# BHEL-Financial-Statement-Analysis-Ratio-Modelling-FY2021-FY2024-
Financial analysis of Bharat Heavy Electricals Limited (BHEL) covering FY2021–FY2024 using audited consolidated financial statements. Includes ratio analysis, cash flow modelling, profitability, liquidity, leverage, working capital efficiency, ROE/ROCE evaluation, and investment risk assessment.
